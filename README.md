### CRUD DE FUNCIONÁRIOS
Foi criado um CRUD de funcionários da biblioteca o qual cada funcionário possui os seguintes atributos:
    <ul>
        <li>Nome Completo</li>
        <li>CPF</li>
        <li>Data de Nascimento</li>
        <li>Data de Admissão</li>
        <li>Email</li>
        <li>Telefone</li>
        <li>Cargo</li>
        <li>Status</li>
    </ul>
